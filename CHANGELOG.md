# Changelog

All notable changes to Tinct are documented in this file.

The project is currently in alpha (`0.1.0-dev`). During beta prep, we document all user-visible changes here before release.

## [Unreleased]

### Added
- `CONTRIBUTING.md` with setup steps, local quality gates, and pull request workflow.
- `CODE_OF_CONDUCT.md` for community behavior and enforcement expectations.
- `SECURITY.md` with private vulnerability reporting and response timelines.

### Documentation
- Added `CHANGELOG.md` to track release notes and policy changes.

## [0.1.0-dev]

### Added
- Initial alpha release of Tinct as a pure Elixir terminal UI framework.
- Elm-style component model (`init/update/view`) with command-based side effects.
- Core widgets, headless test helpers, and example applications.
